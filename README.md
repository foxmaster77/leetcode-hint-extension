# 💡 LeetCode Hint Assistant

A Chrome Extension that adds an AI-powered hint button to every LeetCode problem — 
gives you a nudge in the right direction without spoiling the solution or writing the code for you.

## ✨ Features
- Injects a "💡 Get Hint" button on every LeetCode problem page
- Uses Gemini API to generate smart, spoiler-free hints
- Clean dark UI that matches LeetCode's vibe
- Works instantly — no login, no setup

## 🛠️ Tech Stack
- Chrome Extension (Manifest V3)
- Vanilla JavaScript
- Google Gemini API (gemini-1.5-flash)

## 🚀 Installation
1. Clone this repo
2. Add your Gemini API key in `content.js`
3. Go to `chrome://extensions/` → Enable Developer Mode
4. Click **Load unpacked** → select this folder
5. Open any LeetCode problem and click **💡 Get Hint**
